# Mein Scope — Contact / Request a Demo Page Copy (Draft 1)

Job of this page: the single conversion point every CTA on the site routes to. No live chat exists yet, so this is a form (or mailto) that sets expectations on response time.

---

## Header

**Headline:**
> See Mein Scope on your own device

**Subheadline:**
> Tell us a bit about your clinic and we'll set up a walkthrough — including offline, if you want to see it for yourself.

---

## Form fields (suggested)

- Name
- Clinic / hospital name
- Email
- Phone (optional)
- Country
- Number of clinicians
- What are you using for intake/reporting today? (open text)
- Anything else we should know? (open text)

**Submit button:** `Request a Demo`

**Post-submit message:**
> Thanks — we'll get back to you within [X business days] to set up a time.

*Annotation: fill in the actual response-time commitment once there's a real process behind it — don't publish a number until it's true.*

---

## Alternate / fallback

If a form isn't ready for launch, a direct mailto works as a placeholder:

> Email us at **[hello@meinscope.app]** (placeholder — confirm real address) and tell us a bit about your clinic. We'll follow up to set up a demo.

*Annotation:* Keeping this page light on required fields lowers friction for a first contact — deeper qualification (procedure volume, single vs. multi-site) can happen in the follow-up conversation instead of the form.
